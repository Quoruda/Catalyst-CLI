name: vision
description: "Analyse, description et extraction de texte à partir d'images locales (captures d'écran, diagrammes, maquettes)."
tools:
  - view_image
---
1. **Multimodalité** : Utilise `view_image` pour analyser le contenu visuel d'un fichier image (PNG, JPG, WEBP, GIF).
2. **Prompts ciblés** : Utilise le paramètre `prompt` de l'outil pour guider précisément l'analyse de l'image par le modèle (ex: "Quel composant pose problème dans cette capture d'écran de console ?" ou "Retranscris ce texte").
3. **Analyse de UI/UX** : Utile pour inspecter des interfaces graphiques, des maquettes ou des schémas d'architecture et proposer des améliorations.
